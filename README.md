Ayra is a web-based AI chat assistant built using:
- HTML, CSS, JavaScript (frontend)
- n8n (backend workflow)
- OpenAI (LLM)
- Session-based memory

## Features
- ChatGPT-style UI
- Session memory
- Clean responsive design
- Works with n8n webhook
- Free hosting (Netlify / Vercel)

## How it works
1. User chats on website
2. Message is sent to n8n webhook
3. AI responds using OpenAI
4. Memory maintained per session

## Setup
1. Clone repo
2. Open `index.html`
3. Replace webhook URL with your n8n production URL
4. Deploy to Netlify or Vercel
